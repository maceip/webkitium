# Platform Mascot Images

Place the platform-specific mascot artwork files here:

- `android.png` — Android mascot (green theme, headphones)
- `ios.png` — iOS mascot (blue eyes, white theme)
- `linux.png` — Linux mascot (penguins, terminal background)
- `macos.png` — macOS mascot (blue hair, space background)
- `windows.png` — Windows mascot (pink hair, blue theme)

These images are loaded as the default homepage content in each platform's browser shell.
Desktop shells (macOS, Linux, Windows) center the image horizontally with padding.
Mobile shells (iOS, Android) fit the image to screen width.
